## Start

### Create repo

```bash
gh repo create chicks-net/just-demo-pre1 --private --description "demo of just" --readme
gh repo clone chicks-net/just-demo-pre1
cd just-demo-pre1
```

### Rust hello world

```bash
cargo init .
cargo run --
```
